## アプリを作ってみる

Flutterを使って様々なアプリを作り、  
iOS/Android/Webアプリとして公開できるようになりましょう 💪

## こんな機能を実装したい

- [Providerで状態管理](/create-app/provider)
- [音楽・動画を再生する](/create-app/media-player)

実際にアプリを作りながら、
