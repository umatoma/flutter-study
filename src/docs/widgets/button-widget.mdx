## FlatButton・OutlineButton・RaisedButton

基本的なボタン表示用Widgetとしては以下の3種類があります。

- FlatButton
    - 影のないボタン
- OutlineButton
    - 枠線のあるボタン
- RaisedButton
    - 影のあるボタン

### ボタンを表示する

各ボタン表示用Widgetの基本的な使い方としては、  
`child` にボタン内に表示したいWidgetを指定し、  
`onPressed` にタップされた時の処理を指定すればOKです 👍

```dart
FlatButton(
  onPressed: () { /* ボタンがタップされた時の処理 */ },
  child: Text('click here'),
)
OutlineButton(
  onPressed: () { /* ボタンがタップされた時の処理 */ },
  child: Text('click here'),
)
RiasedButton(
  onPressed: () { /* ボタンがタップされた時の処理 */ },
  child: Text('click here'),
)
```

### 押せなくする

ボタンを押せないようにする（無効化）する時は、`onPressed` に `null` を指定すればOKです 👍  
ボタンが押せるときと押せないときでデザインが自動的に切り替わったりします。

```dart
FlatButton(
  onPressed: () { /* ボタンが押せる時 */ },
  child: Text('押せるよ'),
)
OutlineButton(
  onPressed: null,
  child: Text('押せないよ'),
)
```

### 背景色を変える

`FlatButton` や `RaisedButton` の背景色を変えるときは `color` を指定すればOKです 👍  
また、ボタンが押せない時の背景色は `disabledColor` を指定すればOKです 👍

```dart
FlatButton(
  color: Colors.blue,
  onPressed: () { /* ボタンが押せる時 */ },
  child: Text('押せるよ'),
)
RaisedButton(
  color: Colors.grey,
  onPressed: null,
  child: Text('押せないよ'),
)
```

### 枠線の色を変える

`OutlineButton` の枠線の色を変えるときは `borderSide` を指定すればOKです 👍  
また、ボタンが押せない時の枠線の色は `disabledBorderColor` を指定すればOKです 👍

```dart
OutlineButton(
  borderSide: BorderSide(color: Colors.blue),
  onPressed: () { /* ボタンが押せる時 */ },
  child: Text('押せるよ'),
)
OutlineButton(
  disabledBorderColor: Colors.red,
  onPressed: null,
  child: Text('押せないよ'),
)
```

### 影の深さを変える

`RaisedButton` の影の深さを変えるときは `elevation` を指定すればOKです 👍

```dart
RaisedButton(
  borderSide: BorderSide(color: Colors.blue),
  onPressed: () { /* ボタンが押せる時 */ },
  child: Text('押せるよ'),
)
```

## IconButton

### アイコンボタンを表示する

アイコンボタンを表示する時は `IconButton` を使えばOKです 👍

```dart
IconButton(
  onPressed: () {},
  // 表示アイコン
  icon: Icon(Icons.thumb_up),
  // アイコン色
  color: Colors.pink,
  // サイズ
  iconSize: 64,
)
```

### テキスト＋アイコンボタンを表示する

テキストとアイコンを含めたボタンを表示する時は、  
`FlatButton.icon`・`OutlineButton.icon`・`RaisedButton.icon` を使えばOKです 👍

基本的な使い方は `FlatButton`・`OutlineButton`・`RaisedButton` と同じですが、  
`icon` でアイコンを指定し、`label` でテキストを指定する部分が異なってます 🤔

```dart
FlatButton.icon(
  onPressed: () {},
  icon: Icon(Icons.thumb_up),
  label: Text('Good'),
)
OutlineButton.icon(
  onPressed: () {},
  icon: Icon(Icons.favorite),
  label: Text('Like'),
)
RaisedButton.icon(
  onPressed: () {},
  icon: Icon(Icons.flight),
  label: Text('Flight'),
)
```

## FloatingActionButton

### FloatingActionButtonを表示する

マテリアルデザインを採用したアプリで表示されるFloatingActionButtonを表示する時は、  
`FloatingActionButton` を使えばOKです 👍

`Scaffold` の `floatingActionButton` に指定することで手軽に表示することができます。

```dart
Scaffold(
  floatingActionButton: FloatingActionButton(
    backgroundColor: Colors.green,
    onPressed: () {},
    child: Icon(Icons.add),
  ),
)
```

## まとめ

- 基本的なボタンは `FlatButton`・`OutlineButton`・`RaisedButton` Widgetを使えばOK
- アイコンボタンは `IconButton` Widgetを使えばOK
- アイコン＋テキストボタンは `FlatButton.icon`・`OutlineButton.icon`・`RaisedButton.icon`  Widgetを使えばOK
- `FloatingActionButton` を使う時は `Scaffold` と組み合わせればOK

次は、リストを表示するためのWidgetを紹介していきます 💪💪💪
